Guia de Integração Física (Arduino)

Carregue o código C++ abaixo no seu Arduino. Conecte um potenciômetro na porta A0 e botões ou botões virtuais nas entradas configuradas para enviar os comandos pela Serial.

Código C++ para o Arduino (.ino):


// ==================================================
// HACKER LOCKPICKING - HARDWARE ARDUINO
// ==================================================
const int PIN_POT = A0;

void setup() {
  Serial.begin(9600);
  pinMode(PIN_POT, INPUT);
}

void loop() {
  // Leitura do potenciômetro (mapeado de 0 a 9)
  int valPot = analogRead(PIN_POT);
  int dialVal = map(valPot, 0, 1023, 0, 9);
  
  // Envia valor atual do dial para o decodificador
  Serial.print("Visor: ");
  Serial.println(dialVal);
  
  // Escuta comando de RESET do navegador
  if (Serial.available() > 0) {
    String input = Serial.readStringUntil('\n');
    input.trim();
    if (input == "RESET") {
      // Reinicia o estado do hardware aqui
    }
  }
  
  delay(100); // 10 FPS
}

Mensagens suportadas pelo protocolo Serial:

  • Visor: X - Atualiza o medidor analógico (X deve ser de 0 a 9).
  • Catraca 1 TRAVADA: X - Grava a Catraca 1 com o valor X.
  • Catraca 2 TRAVADA: X - Grava a Catraca 2 com o valor X.
  • Catraca 3 TRAVADA: X - Grava a Catraca 3 com o valor X.
  • ACESSO LIBERADO - Acende o painel em verde (Senha correta).
  • ACESSO NEGADO - Acende o painel em vermelho (Senha incorreta).
  • Cofre resetado - Reinicia a interface para o modo inicial.
