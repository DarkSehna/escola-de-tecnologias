Guia de Integração Física (Arduino)

Carregue o código C++ abaixo no seu Arduino. Conecte um potenciômetro na porta A0 e botões ou botões virtuais nas entradas configuradas para enviar os comandos pela Serial.

Código C++ para o Arduino (.ino):


// ==================================================
// HACKER LOCKPICKING - HARDWARE ARDUINO
// ==================================================
int pinoBotao = 2; // Botão ligado no pino 2 e GND (INPUT_PULLUP)
int pinoPot = A0;  // Potenciômetro ligado no pino A0

int estadoCofre = 1;

void setup() {
  Serial.begin(9600);
  pinMode(pinoBotao, INPUT_PULLUP);
  
  Serial.println("--- SISTEMA DE SEGURANCA ATIVADO ---");
}

void loop() {
  // 1. Envia o valor atual do dial para o navegador
  int numeroDoCofre = map(analogRead(pinoPot), 0, 1023, 0, 9);
  Serial.print("Visor: ");
  Serial.println(numeroDoCofre);
  
  // 2. Se o botão for pressionado (LOW no modo PULLUP)
  if (digitalRead(pinoBotao) == LOW) {
    // Envia o sinal de trava para a catraca atual
    Serial.print("Catraca ");
    Serial.print(estadoCofre);
    Serial.print(" TRAVADA: ");
    Serial.println(numeroDoCofre);
    
    estadoCofre++;
    
    // Se passar da terceira catraca, reinicia localmente
    if (estadoCofre > 3) {
      estadoCofre = 1;
    }
    delay(400); // Debounce para evitar cliques múltiplos
  }
  
  // 3. ESCUTA comandos de RESET vindos do navegador! (Crucial para sincronizar)
  if (Serial.available() > 0) {
    String input = Serial.readStringUntil('\n');
    input.trim();
    if (input == "RESET") {
      estadoCofre = 1;
      Serial.println("RESET: Hardware resetado com sucesso.");
    }
  }
  
  delay(150); // Controla a velocidade de atualização
}

Mensagens suportadas pelo protocolo Serial:

  • Visor: X - Atualiza o medidor analógico (X deve ser de 0 a 9).
  • Catraca 1 TRAVADA: X - Grava a Catraca 1 com o valor X.
  • Catraca 2 TRAVADA: X - Grava a Catraca 2 com o valor X.
  • Catraca 3 TRAVADA: X - Grava a Catraca 3 com o valor X.
  • ACESSO LIBERADO - Acende o painel em verde (Senha correta).
  • ACESSO NEGADO - Acende o painel em vermelho (Senha incorreta).
  • Cofre resetado - Reinicia a interface para o modo inicial.
